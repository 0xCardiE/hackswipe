## SEMrush

**Use for:** Competitor research, content gap analysis, backlink prospecting.

### Useful workflows

1. **Export top competitor pages**
   See which URLs drive the most organic traffic for each competitor. Use this to spot page types worth copying (Chrome Web Store listing SEO, landing pages, pricing pages).

2. **Export top competitor keywords**
   Pull the keywords competitors rank for overall. Cross-check against [`kws-list.md`](./kws-list.md) — add high-intent terms we missed; deprioritize mismatches (other dating apps, dev/API intent).

3. **Identify top pages to build + export their ranking keywords**
   - In SEMrush: Organic Research → Pages (per competitor). Sort by traffic or keyword count.
   - Shortlist URLs we could realistically beat with Tinder-focused, honestly-positioned content.
   - Map exports to our site structure: homepage, `/guides/*`, `/compare/*`, FAQ, pricing.

4. **Export competitor backlinks**
   Use for outreach and placement ideas: "best Tinder auto swiper" roundups, dating-app-hack listicles, Chrome extension roundups.

## Chrome Web Store listing research

**Use for:** Since most competitors live primarily as Chrome Web Store listings (not full marketing sites), the store listing itself is often the main ranking asset to study.

- Check listing titles, short/long descriptions, screenshots, and review count/rating for each competitor.
- Note which competitors have their own marketing site vs store-listing-only (informs whether a compare page or a Chrome Web Store optimization pass is the better move).
